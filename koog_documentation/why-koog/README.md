<h1 id="why-koog">Why Koog</h1>
<p>Koog is designed to solve real-world problems with JetBrains-level quality.
It provides advanced <abbr title="Artificial Intelligence">AI</abbr> algorithms, out-of-the-box proven techniques, a <abbr title="Kotlin-based domain-specific language">Kotlin DSL</abbr>, as well as a Java fluent <abbr title="Application Programming Interface">API</abbr>, and robust multi-platform
support that goes beyond traditional frameworks. </p>
<p>Its primary focus is reliability — enabling <abbr title="Artificial Intelligence">AI</abbr> agents that can be confidently used in demanding enterprise environments.</p>
<h2 id="integration-with-java-and-kotlin-applications">Integration with Java and Kotlin applications</h2>
<p>Koog provides a Kotlin Domain-Specific Language (DSL) designed specifically for Kotlin developers, along with a fluent Java <abbr title="Application Programming Interface">API</abbr> for Java users.
The same framework delivers a native feel in both <abbr title="Java Virtual Machine">JVM</abbr> languages, ensuring seamless integration into Kotlin and Java applications
while significantly boosting productivity and enhancing the overall developer experience.</p>
<h2 id="real-world-validation-with-jetbrains-products">Real-world validation with JetBrains products</h2>
<p>Koog powers multiple JetBrains products, including internal <abbr title="Artificial Intelligence">AI</abbr> agents. 
This real-world integration ensures that Koog is continuously tested, refined, and validated against practical use cases. 
It focuses on what works in practice, incorporating insights from extensive feedback and real-product scenarios. 
This integration provides Koog with strengths that distinguish it from other frameworks.</p>
<h2 id="advanced-solutions-available-out-of-the-box">Advanced solutions available out of the box</h2>
<p>Koog includes pre-built, composable solutions to simplify and speed up the development of agentic systems, setting it apart from frameworks that only offer basic components:</p>
<ul>
<li><strong>Graph workflows with domain modeling.</strong> Model <abbr title="Artificial Intelligence">AI</abbr> workflows as explicit graphs built on verified domain models. By expressing requirements as structured data classes instead of relying on naive prompting, you gain precise control over agent behavior and significantly improve reliability and predictability.</li>
<li><strong>Multiple history compression strategies.</strong> Koog comes with advanced strategies to compress and manage long-running conversations out of the box, eliminating the need for manual experimentation with approaches. With fine-tuned prompts, techniques, and algorithms tested and refined by <abbr title="Machine Learning">ML</abbr> engineers, you can rely on proven methods to improve performance. For more details on compression strategies, refer to <a href="https://docs.koog.ai/history-compression/">History compression</a>. To explore how Koog handles compression and context management in real-world scenarios, check out <a href="https://blog.jetbrains.com/ai/2025/07/when-tool-calling-becomes-an-addiction-debugging-llm-patterns-in-koog/">this article</a>.</li>
<li><strong>Advanced persistence (Durable execution).</strong> Koog lets you restore full agent state machines instead of just chat messages. This enables features like checkpoints, failure recovery, and even the ability to revert to any point in the execution of the state machine.</li>
<li><strong>All modern agent patterns, one framework.</strong> Graph workflows, <abbr title="Goal-Oriented Action Planning">GOAP</abbr> (Goal-Oriented Action Planning) and <abbr title="Large Language Model">LLM</abbr> planning, multi-agent orchestration — fully supported and fully composable. Build exactly the agent your use case requires.</li>
<li><strong>Seamless <abbr title="Large Language Model">LLM</abbr> switching.</strong> You can switch a conversation to a different large language model (<abbr title="Large Language Model">LLM</abbr>) with a new set of available tools at any point without losing the existing conversation history. Koog automatically rewrites the history and handles unavailable tools, enabling smooth transitions and a natural interaction flow.</li>
<li><strong>Robust retry components.</strong> Koog includes a retry mechanism that lets you wrap any set of operations within your agentic system and retry them until they meet configurable conditions. You can provide feedback and adjust each attempt to ensure reliable results. If <abbr title="Large Language Model">LLM</abbr> calls time out, tools do not work as expected, or there are network issues, Koog ensures that your agent remains resilient and performs effectively, even during temporary failures. For more technical details, see <a href="https://docs.koog.ai/history-compression/">Retry functionality</a>.</li>
</ul>
<h2 id="broad-integration-multiplatform-support-enhanced-observability">Broad integration, multiplatform support, enhanced observability</h2>
<p>Koog supports the development and deployment of agentic applications across a variety of platforms and environments:</p>
<ul>
<li><strong>Spring Boot, Spring <abbr title="Artificial Intelligence">AI</abbr> and Ktor integrations</strong>. Koog integrates with widely used enterprise environments.<ul>
<li>For Spring Boot, Koog provides ready-to-use beans and auto-configured <abbr title="Large Language Model">LLM</abbr> clients, making it easy to start building <abbr title="Artificial Intelligence">AI</abbr>-powered workflows.</li>
<li>If you're already using Spring <abbr title="Artificial Intelligence">AI</abbr> for <abbr title="Large Language Model">LLM</abbr> and <abbr title="Retrieval-Augmented Generation">RAG</abbr> capabilities, Koog can be layered on top as an orchestration and agentic framework. This allows you to leverage Spring <abbr title="Artificial Intelligence">AI</abbr>’s extensive integrations while benefiting from Koog’s advanced, reliable, and cost-efficient <abbr title="Artificial Intelligence">AI</abbr> workflows.</li>
<li>If you have a Ktor server, you can install Koog as a plugin, configure providers using configuration files, and call agents directly from any route without manually connecting <abbr title="Large Language Model">LLM</abbr> clients.</li>
</ul>
</li>
<li><strong>Multiplatform support</strong>. You can deploy your agentic applications across <abbr title="Java Virtual Machine">JVM</abbr>, <abbr title="JavaScript">JS</abbr>, WasmJS, Android, and iOS targets.</li>
<li><strong>Broad <abbr title="Artificial Intelligence">AI</abbr> integration</strong>. Koog integrates with major <abbr title="Large Language Model">LLM</abbr> providers, including OpenAI, Anthropic, Google, DeepSeek, Mistral, Alibaba, as well as enterprise-level <abbr title="Artificial Intelligence">AI</abbr> clouds like Bedrock. It also supports local models such as Ollama. For the full list of available providers, see <a href="https://docs.koog.ai/llm-providers/"><abbr title="Large Language Model">LLM</abbr> providers</a>.</li>
<li><strong>OpenTelemetry support</strong>. Koog provides out-of-the-box integration with popular observability providers like <a href="https://wandb.ai/site/weave/">W&amp;B Weave</a>, <a href="https://langfuse.com/">Langfuse</a> and <a href="https://www.datadoghq.com/">DataDog</a> for monitoring and debugging <abbr title="Artificial Intelligence">AI</abbr> applications. With native OpenTelemetry support, you can trace, log, and measure your agents using the same tools you already use in your system. To learn more, refer to <a href="https://docs.koog.ai/opentelemetry-support/">OpenTelemetry</a>.</li>
</ul>
<h2 id="collaboration-with-ml-engineers-and-product-teams">Collaboration with <abbr title="Machine Learning">ML</abbr> engineers and product teams</h2>
<p>A unique advantage of Koog is its direct collaboration with JetBrains <abbr title="Machine Learning">ML</abbr> engineers and product teams.
This ensures that features built with Koog are not just theoretical but tested and refined based on real-world product requirements.
This means that Koog incorporates:</p>
<ul>
<li><strong>Fine-tuned prompts and strategies</strong> optimized for real-world performance.</li>
<li><strong>Proven engineering approaches</strong> discovered and validated through product development, such as its unique history compression strategies. You can learn more in <a href="https://blog.jetbrains.com/ai/2025/07/when-tool-calling-becomes-an-addiction-debugging-llm-patterns-in-koog/">this detailed article</a>.</li>
<li><strong>Continuous improvements</strong> that help Koog stay efficient and adaptable to evolving needs.</li>
</ul>
<h2 id="commitment-to-the-developer-community">Commitment to the developer community</h2>
<p>The Koog team is deeply committed to building a strong developer community.
By actively gathering and incorporating feedback, Koog evolves to meet the needs of developers effectively.
We are actively expanding support for diverse <abbr title="Artificial Intelligence">AI</abbr> architectures, comprehensive benchmarks, detailed use-case guides,
and educational resources to empower developers.</p>
